# Quick Start Guide

## 2-Minute Setup

1. **Clone the repository:**  
   Open your terminal and run:
   ```bash
   git clone https://github.com/<owner>/Ai-help.git
   cd Ai-help
   ```  

2. **Install dependencies:**  
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```  

3. **Run the application:**  
   Execute the following command to start the app:
   ```bash
   npm start
   ```  

You're all set! Your application should now be running.

For further documentation, please refer to the [Documentation](LINK_TO_DOCUMENTATION).  

Enjoy!
